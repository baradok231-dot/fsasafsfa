# BotHub - Telegram Bot Aggregator

Универсальный Telegram бот с 30+ инструментами в одном месте.

## Быстрый старт

```bash
# 1. Скачайте папку bot-standalone
# 2. Запустите бота:
node bot.js
```

Или с переменной окружения:
```bash
export TELEGRAM_BOT_TOKEN="ВАШ_ТОКЕН"
node bot.js
```

## Функции

### Работают сразу (без API):
- QR-код генератор
- Погода (wttr.in)
- Калькулятор
- Рандомайзер (числа, выбор, монетка)
- Генератор паролей
- Конвертер валют

### Требуют API:
- Скачивание видео (TikTok, YouTube, Instagram, Twitter)
- Удаление фона
- AI чат (ChatGPT)
- Генерация изображений
- Переводчик
- Поиск музыки

## Категории

- **Медиа** — скачивание видео и музыки
- **Инструменты** — работа с фото и файлами
- **AI / Нейросети** — ChatGPT и генерация
- **Соцсети** — контент для соцсетей
- **Утилиты** — погода, валюты, калькулятор
